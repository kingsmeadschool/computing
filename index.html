Task 2: HTML

HTML stands for Markup Language. It is a standard markup language used for creating and structuring on the web. HTML uses tags to define the structure and of webpages, including headings, paragraphs, lists, links, images, and more. It is the foundation of most websites and is used with other web technologies like CSS and JavaScript to create dynamic, interactive .
